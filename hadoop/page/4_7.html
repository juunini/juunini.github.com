Accumulo Apache Accumulo® is a sorted, distributed key/value store that provides robust, scalable data storage and retrieval. Accumulo는 정돈된 분산 key/value를 건강하고 사용 가능한 데이터 저장소와 검색을 가능하게 한다. Apache Atlas Data Governance and Metadata framework for Hadoop Hadoop의 데이터 관리 및 메타 데이터 프레임 워크 Druid Druid is a high-performance, column-oriented, distributed data store. Druid는 고성능의 column 기반 분산 데이터 저장소입니다. Apache Falcon Feed management and data processing platform 피드 관리 및 데이터 처리 플랫폼 Apache Flume Flume is a distributed, reliable, and available service for efficiently collecting, aggregating, and moving large amounts of log data. It has a simple and flexible architecture based on streaming data flows. It is robust and fault tolerant with tunable reliability mechanisms and many failover and recovery mechanisms. It uses a simple extensible data model that allows for online analytic application. Flume은 대량의 로그 데이터를 효율적으로 수집, 집계 및 이동하기위한 분산 형의 안정적이고 사용 가능한 서비스입니다. 스트리밍 데이터 흐름을 기반으로하는 간단하고 유연한 아키텍처입니다. 조정 가능한 신뢰성 메커니즘과 많은 장애 조치 및 복구 메커니즘으로 견고하고 내결함성이 있습니다. 온라인 분석 응용 프로그램을 허용하는 단순한 확장 가능 데이터 모델을 사용합니다. Apache HBase Apache HBase™ is the Hadoop database, a distributed, scalable, big data store. Apache HBase™는 분산되고 확장 가능한 대용량 데이터 저장소 인 Hadoop 데이터베이스입니다. APACHE HIVE The Apache Hive ™ data warehouse software facilitates reading, writing, and managing large datasets residing in distributed storage using SQL. Structure can be projected onto data already in storage. A command line tool and JDBC driver are provided to connect users to Hive. Apache Hive ™ 데이터웨어 하우스 소프트웨어는 SQL을 사용하여 분산 스토리지에있는 대규모 데이터 세트의 읽기, 쓰기 및 관리를 용이하게합니다. 구조는 이미 스토리지에있는 데이터에 투영 될 수 있습니다. 사용자를 Hive에 연결하기 위해 명령 줄 도구와 JDBC 드라이버가 제공됩니다.
